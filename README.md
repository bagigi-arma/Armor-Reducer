# Bagigi's Armor Reducer

This is a mod for Arma 3 made by the [Bagigi Arma 3 Community](https://units.arma3.com/unit/bagigi), with the purpose of tweaking the config to reduce the effectiveness of RHS body armor to gameplay-friendly levels for both TvE and TvT.

It is available [on the Steam Workshop](https://steamcommunity.com/sharedfiles/filedetails/?id=3593413120).

## Support

You can build this mod with the [HEMTT build tool](https://hemtt.dev/index.html).
